积木使用小技巧

发表于 2018-06-06 | 分类于 个人项目 | | 阅读次数

可以在 “积木” -> “更多” -> “小技巧” 找到这篇文章的入口。

有任何问题、建议或 Bug 都可以点击 “积木” 中的 “更多” -> “联系我” 来给我发邮件。或者直接发送到 codingallnight@gmail.com。

或者可以在微博上 at 我 @我的眼里只有代码。

English

0x00 小技巧列表

  • 概览界面
  • 计算器界面
  • 创建 / 编辑交易
  • 其他

0x01 概览界面

  • 总余额（金额）双击隐藏或展示
  • 点击账户（列表）进入所有账户列表
  • 在概览界面摇晃手机，打开新增交易界面（后期可配置）

0x02 计算器界面

  • 点击“等号”计算结果或者返回编辑界面

0x03 创建 / 编辑交易

  • 长按顶部金额视图会重置金额为 0
  • 再次点击已选中的分类取消选中
  • 长按日期重置为当前日期
  • 长按时间重置为当前时间
  • 在交易列表界面摇晃手机，打开新增交易界面（后期可配置）

0x04 其他

  • 日历、交易列表和预算交易列表界面支持 3D Touch 预览
  • 软件图标 3D Touch 快速新建交易
